Mirror for Photoshop

Real-time Photoshop preview on iOS

Stream your Photoshop artboards directly to your iPhone or iPad. See exactly how your designs look on the actual device as you work.

App Store Photoshop Plugin Support Privacy

Install the Server

macOS

Homebrew(recommended)
brew install hex/mirror-for-photoshop/mirror-for-photoshop-server
Run
ps-mirror-server

Or run as a background service: brew services start mirror-for-photoshop-server

Alternatively, download the standalone app from GitHub Releases.

Windows

Winget(recommended)
winget install Hex.MirrorForPhotoshopServer

Or download from GitHub Releases.
